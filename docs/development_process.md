# Development Process

## Tools & Workflow
- Visual Studio Code for editing
- Git for version control
- GitHub for remote hosting and GitHub Pages deployment
- Live Server VS Code extension for preview

## Steps followed
1. Created project skeleton (index.html, styles/style.css).
2. Built header and navigation.
3. Styled layout with CSS using Flexbox and Grid.
4. Tested responsiveness and fixed layout issues for small screens.
5. Initialized git, committed changes progressively.
6. Pushed repo to GitHub and enabled GitHub Pages.

## Commands used
git init
git add .
git commit -m "Initial commit"
git remote add origin https://github.com/nitin-gahlot/CS-ASSIGNMENT-04.git
git push -u origin main
